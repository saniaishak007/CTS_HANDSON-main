### 50. Initialize Flask-Migrate.

Initialize the migration repository.

```bash
flask db init
```

Generate migration.

```bash
flask db migrate -m "Initial Schema"
```

Apply migration.

```bash
flask db upgrade
```

Verify the tables are created successfully.


---

### 51. Insert Sample Data using Flask Shell.

Open Flask shell.

```bash
flask shell
```

Import models.

```python
from courses.models import *
```

Create departments.

```python
d1 = Department(name="Computer Science")
d2 = Department(name="Electronics")

db.session.add(d1)
db.session.add(d2)
db.session.commit()
```

Create courses.

```python
c1 = Course(name="Python", code="CS101", credits=4)
c2 = Course(name="Flask", code="CS102", credits=3)
c3 = Course(name="Database", code="CS103", credits=4)

db.session.add_all([c1, c2, c3])
db.session.commit()
```

Verify.

```python
Course.query.all()
```

Exit Flask shell.

```python
exit()
```
